qgisMinimumVersion=3.28

description=Install the GDAL H2GIS driver to read/write H2GIS spatial databases (.mv.db) in QGIS
about=This plugin automates the installation of the native GDAL/OGR H2GIS driver. 
    It detects your QGIS GDAL version, downloads the matching pre-compiled binary, 
    and configures everything automatically. Also includes a demo H2GIS database 
    to get started quickly.
    
    Features:
    - Automatic OS and GDAL version detection
    - Downloads matching pre-compiled driver binary
    - Installs H2GIS native library
    - Creates sample demo database
    - No admin rights required (user installation)
    
    After installation, you can open .mv.db files directly in QGIS!
