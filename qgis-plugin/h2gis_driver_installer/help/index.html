üî∑ What is H2GIS?

H2GIS is a spatial extension of the H2 database engine. It provides:

  • Full SQL support with spatial functions
  • Embedded database (no server needed)
  • Fast performance for GIS operations
  • Used by NoiseModelling and other GIS applications

üì• Installation

Step 1: Open the Installer

Go to Database ‚Üí H2GIS Driver Installer

Step 2: Check Platform Info

The dialog shows your detected platform information:

  • OS: Linux, Windows, or macOS
  • Architecture: x86_64 or arm64
  • GDAL Version: Your installed GDAL version

Step 3: Install

Click Install Driver and wait for the download and installation to complete.

Step 4: Restart QGIS

Important: You must restart QGIS for the driver to be loaded.

üöÄ Usage

Opening H2GIS Files

After installation, you can open .mv.db files like any other data source:

  1. Use Layer ‚Üí Add Layer ‚Üí Add Vector Layer
  2. Or drag and drop the .mv.db file into QGIS
  3. Or use the Browser panel

Demo Database

Click Create Demo Database to generate a sample H2GIS database with:

  • cities - Major French cities (points)
  • rivers - Major rivers (lines)
  • france_boundary - Country outline (polygon)

üîß Troubleshooting

Driver not found after restart

Make sure the QGIS startup script is properly configured. The plugin creates qgis_h2gis_startup.py in your QGIS profile directory.

Download failed

Check your internet connection. If behind a proxy, configure it in QGIS settings.

Unsupported platform

Pre-compiled binaries are available for common configurations. For other platforms, you may need to compile from source.
